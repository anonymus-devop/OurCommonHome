Instalar la App 🌿

Guía fácil paso a paso

1. Descargar la app

Presiona el botón para descargar la aplicación en tu celular.

2. Permitir instalación

Si aparece un mensaje de seguridad:

👉 Pulsa "Configuración"
👉 Activa "Permitir desde esta fuente"
👉 Regresa

3. Instalar

Abre el archivo descargado y pulsa "Instalar".

4. Abrir la app

Una vez instalada, pulsa "Abrir" y listo 🎉
